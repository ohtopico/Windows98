%Os <b>programas</b> são séries organizadas de instruções que, quando executadas, fazem com que o computador se comporte de uma determinada maneira. O WordPad, por exemplo, é um programa.
%|%A <b>barra de tarefas</b> é a barra retangular cinza normalmente localizada na parte inferior de sua tela. A barra de tarefas inclui o botão <b>Iniciar</b>, o relógio do computador e um botão para cada programa que estiver aberto no momento. Ela exibe também barras de ferramentas.
%|%A <b>barra de título </b> é a barra horizontal localizada na parte superior de uma janela que normalmente exibe o nome do documento ou do programa que aparece na janela. A barra de título também contém o ícone do programa e os botões <b>Maximizar</b>, <b>Minimizar</b> e <b>Fechar</b>.
%|%Um <b>arquivo</b> é um conjunto de textos ou de dados gravados sob um nome exclusivo. A maioria das informações armazenadas em seu computador encontra-se armazenada em arquivos.
%|%Uma <b>pasta</b> é um recipiente para programas, arquivos e outras pastas utilizadas para organizar informações em seu computador. No Windows, uma pasta é representada pelo ícone de uma pasta de arquivos. As pastas são o equivalente aos diretórios em alguns outros sistemas operacionais para computadores.
%|%Utilizado para salvar dados do computador, o <b>disco rígido </b> é composto de discos magnéticos e redondos, não-flexíveis, empilhados dentro de um recipiente metálico lacrado. Os dados são armazenados em ambos os lados dos discos. A maioria dos discos rígidos tem entre dois e oito discos.
%|%Para selecionar um arquivo ou uma pasta, clique nele. Para abrir um arquivo ou uma pasta, clique sobre ela duas vezes.
%|%Você clica com o <b>botão direito</b> do mouse em um item colocando o ponteiro do mouse sobre o item e pressionando e soltando o botão rapidamente. Clicar com o botão direito do mouse em um item exibe um menu de atalho com comandos relacionados ao item.
%|%<b>Arrastar</b> é o procedimento executado para mover ou copiar um objeto de um lugar para outro na tela utilizando-se o mouse. Clique e mantenha pressionado o botão do mouse quando o ponteiro estiver sobre o objeto, mova o ponteiro para o novo local e solte o botão do mouse.
%|%Você também pode utilizar os botões <b>Recortar</b> e <b>Colar</b>, na barra de ferramentas, para mover arquivos entre pastas.
%|%Uma <b>caixa de diálogo</b> é um tipo de janela exibida pelo sistema operacional Windows ou por um programa; ela apresenta informações ou solicita que você as insira.
%|%<b>Ativo</b> é o estado do item (como uma janela, ícone, guia ou botão) que você seleciona utilizando o mouse ou o teclado.
%|%<b>Padrão</b> refere-se a um valor ou configuração que um dispositivo ou programa seleciona automaticamente se você não especificar outro.
%|%Um item ou seção de texto <b>selecionado</b> é uma área na qual o Windows ou um programa irão atuar quando você emitir o próximo comando. Para selecionar uma seção de texto, arraste o ponteiro sobre a seção para destacá-la. Para selecionar um item, como um ícone, clique no item.
%|%Caso seu computador esteja configurado para exibir extensões de arquivo, você precisará selecionar o nome inteiro do arquivo, <b>Documento.doc</b>.
%|%Na caixa <b>Nome:</b> você pode também digitar apenas parte do nome do arquivo que está tentando localizar.
%|%Uma <b>janela</b> é uma área móvel e redimensionável na qual as informações são exibidas e com a qual você pode interagir.
%|%Um <b>ícone</b> é uma pequena imagem gráfica que aparece em sua área de trabalho ou dentro de um programa e que representa um arquivo, uma pasta, um comando, um programa ou uma página da Web.
%|%Sua <b>área de trabalho</b> é a área de trabalho que preenche a tela de seu monitor e exibe ícones, componentes da área de trabalho, caixas de diálogo, janelas e outros elementos. A área de trabalho do Windows é um exemplo de uma interface gráfica de usuário.
%|%É possível mover qualquer janela aberta na área de trabalho arrastando a barra de título da janela. Se você não consegue arrastar a janela, ela pode estar maximizada. Se estiver, clique no botão <b>Restaurar</b>.
%|%A <b>barra de tarefas</b> é a barra retangular cinza normalmente localizada na parte inferior de sua tela. A barra de tarefas inclui o botão <b>Iniciar</b>, o relógio do computador e um botão para cada programa que estiver aberto no momento. Ela também exibe barras de ferramentas.
%|%Você pode também alternar entre programas abertos mantendo pressionada a tecla ALT e pressionando TAB.
%|%Ao <b>minimizar</b> uma janela, você a esconde na área de trabalho, mas o botão que a representa permanece na barra de tarefas. Um programa continua a ser executado enquanto sua janela está minimizada.
%|%Ao <b>maximizar</b> uma janela, ela se expande para preencher a tela inteira ou, em alguns casos, para preencher uma janela maior.

%|%Um <b>link</b> é uma seção de texto ou um gráfico em que você pode clicar para ir a outro local, como uma página da Web, um arquivo ou um local específico em um arquivo. O objeto vinculado está conectado ao novo arquivo ou página da Web pelo código HTML.

%|%Um <b>navegador</b> da Web é um programa (como o Microsoft Internet Explorer) que permite visualizar informações em seu computador, em uma rede ou na World Wide Web. O navegador da Web converte o HTML, a linguagem de computador para a Web, para um formato gráfico. Para que o navegador da Web visualize informações na Web, o computador deve ter estabelecido uma conexão com a Internet.
